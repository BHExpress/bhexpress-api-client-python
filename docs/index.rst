BHExpress es un proyecto que tiene por objetivo gestionar Boletas de Honorarios Electrónicas, utilizando funcionalidades del Servicio de Impuestos Internos (SII) de Chile.

Aquí podrás encontrar la biblioteca escrita en lenguaje PHP para la integración con el Servicio de Impuestos Internos (SII) asociada la creación, manejo y anulación de BHEs.
